# portage-agent (backend)

Python backend for **Portage** — the autonomous code-migration agent. FastAPI control
plane + LangGraph agent worker, checkpointed to Postgres. See the repo root `CLAUDE.md`
and `code-migration-agent-plan.md` for the full architecture and phase plan.

Package: `portage_agent` (the bare name `portage` is taken on PyPI).
